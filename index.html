1. What is HTML?

HTML (HyperText Markup Language) is the foundation of web pages. It uses tags to define the structure and content, such as headings, paragraphs, and links, on the internet.

Developed by Tim Berners-Lee in 1989, HTML has evolved over time. HTML5, released in 2014, introduced features like multimedia support, semantic elements, and new APIs.

2. HTML Structure and Core Elements

2.1 Basic Structure

Every HTML page follows this structure:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is my first webpage.</p>
</body>
</html>
  • <!DOCTYPE html>: Declares the document as HTML5. <!DOCTYPE html>
  • <html>: The root element of the page. <html>
  • <head>: Contains metadata and the title. <head>
  • <body>: Holds the visible content. <body>

2.2 Core Elements

Tag Purpose
<h1> to <h6> Headings of different levels
<p> Paragraphs of text
<a> Links to other pages
<img> Embeds images
<ul>, <ol>, <li> Lists and list items

Example of using elements together:

<h1>My Website</h1>
<p>Welcome! Visit my <a href="https://example.com">blog</a>.</p>
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

3. Attributes and Their Uses

Attributes add extra information to HTML elements and are written as name-value pairs in the opening tag.

Attribute Purpose
id Unique identifier for an element
class Groups elements for styling
href URL for links in <a>
src Source file for images or media
alt Alternative text for images

Example of attributes in action:

<a href="https://example.com" target="_blank">Visit Example</a>
<img src="photo.jpg" alt="A scenic view" width="200">
<p id="intro" class="highlight">Welcome to my page!</p>

4. Forms and Interactivity

Forms allow users to input data, like text or selections, which can be sent to a server.

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>
  <button type="submit">Submit</button>
</form>
  • <form>: Container for form elements. <form>
  • <input>: Input fields for data. <input>
  • required: Makes a field mandatory. required

4.1 More Form Elements

<form action="/contact" method="post">
  <label for="topic">Topic:</label>
  <select id="topic" name="topic">
    <option value="support">Support</option>
    <option value="feedback">Feedback</option>
  </select>
  <label for="message">Message:</label>
  <textarea id="message" name="message" rows="3"></textarea>
  <button type="submit">Send</button>
</form>

5. Semantic HTML and Accessibility

Semantic HTML uses meaningful tags to improve structure, accessibility, and SEO.

Tag Purpose
<header> Introductory content or navigation
<main> Main content of the page
<footer> Footer with copyright info

Accessibility tip: Always use alt for images and aria-label for buttons.

<header>
  <h1>My Site</h1>
</header>
<main>
  <p>Main content here.</p>
</main>
<footer>
  <p>© 2025</p>
</footer>

6. Media and Metadata

6.1 Adding Media

HTML5 supports images, audio, and video:

<img src="image.jpg" alt="A sunset" width="200">
<audio controls>
  <source src="music.mp3" type="audio/mpeg">
</audio>
<video controls width="200">
  <source src="video.mp4" type="video/mp4">
</video>

6.2 Metadata for SEO

Metadata helps search engines understand your page:

<head>
  <meta charset="UTF-8">
  <meta name="description" content="Learn HTML basics">
  <meta name="keywords" content="HTML, web">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>HTML Guide</title>
</head>

7. Advanced Tables and Attributes

HTML tables can be enhanced with additional attributes for better structure and accessibility.

Attribute Purpose
colspan Spans a cell across multiple columns
rowspan Spans a cell across multiple rows
scope Defines the scope of a header cell (row or col)

Example of an advanced table:

<table border="1">
  <tr>
    <th scope="col">Name</th>
    <th scope="col">Category</th>
    <th scope="col">Details</th>
  </tr>
  <tr>
    <td>John</td>
    <td rowspan="2">Student</td>
    <td>Grade A</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>Grade B</td>
  </tr>
  <tr>
    <td colspan="3">Summary: 2 Students</td>
  </tr>
</table>

8. HTML5 APIs and Features

HTML5 introduced APIs and features to enhance web interactivity and functionality.

8.1 Canvas for Graphics

The <canvas> element allows dynamic drawing using JavaScript.

<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
  const canvas = document.getElementById("myCanvas");
  const ctx = canvas.getContext("2d");
  ctx.fillStyle = "blue";
  ctx.fillRect(50, 20, 100, 60);
</script>

8.2 Geolocation API

The Geolocation API retrieves the user's location (with permission).

<button onclick="getLocation()">Get Location</button>
<p id="location"></p>
<script>
  function getLocation() {
    if (navigator.geolocation) {
      navigator.geolocation.getCurrentPosition(showPosition);
    } else {
      document.getElementById("location").innerHTML = "Geolocation not supported.";
    }
  }
  function showPosition(position) {
    document.getElementById("location").innerHTML = 
      "Latitude: " + position.coords.latitude + 
      "<br>Longitude: " + position.coords.longitude;
  }
</script>

8.3 Drag and Drop

HTML5 supports drag-and-drop functionality for interactive elements.

<div id="drag" draggable="true" ondragstart="drag(event)">Drag me</div>
<div id="drop" ondrop="drop(event)" ondragover="allowDrop(event)">Drop here</div>
<script>
  function allowDrop(ev) {
    ev.preventDefault();
  }
  function drag(ev) {
    ev.dataTransfer.setData("text", ev.target.id);
  }
  function drop(ev) {
    ev.preventDefault();
    const data = ev.dataTransfer.getData("text");
    ev.target.appendChild(document.getElementById(data));
  }
</script>

9. Inline Styling and Deprecated Tags

9.1 Inline Styling

The style attribute allows inline CSS for quick styling, though external CSS is preferred for maintainability.

<p style="color: red; font-size: 18px;">This text is red and larger.</p>
<div style="background-color: #f0f0f0; padding: 10px;">A styled div.</div>

9.2 Deprecated Tags

Some HTML tags are deprecated in HTML5 and should be replaced with CSS or semantic alternatives.

Tag Alternative
<font> Use CSS color and font-family
<center> Use CSS text-align: center
<u> Use CSS text-decoration: underline
